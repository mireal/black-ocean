## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mireal/black-ocean.git
```
2. Navigate to the project directory:
```bash
cd black-ocean
```
3. Install the dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```

5. Open your browser and visit:
```bash
http://localhost:3000.
```
